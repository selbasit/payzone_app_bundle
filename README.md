# Pay Zone, Water Saturation, and Shale Volume Predictor (Streamlit)

This app loads LAS-text or CSV well logs and predicts:
- **Pay Zone** (classification)
- **Water Saturation (Sw)** (regression)
- **Shale Volume (Vsh)** (regression)

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
